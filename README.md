Full Stack Realtime Chat App 

### Application link
https://fullstack-realtime-chat-app-b7cy.onrender.com


### Setup .env file

```js
MONGODB_URI=
PORT=5001
JWT_SECRET=

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

NODE_ENV=development
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
